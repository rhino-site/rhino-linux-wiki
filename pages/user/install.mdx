# Installation Guide

Here is the official installation guide for Rhino Linux. This guide assumes that you have created a bootable USB drive with a Rhino Linux disk image. If you have not, you can find guides online for how to create a Live bootable Linux USB on Windows, macOS or Linux. You can grab the latest Rhino Linux disk image [here](https://rhinolinux.org/download).

The following pages will provide instructions for multiple platforms.
